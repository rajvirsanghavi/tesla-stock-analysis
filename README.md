# Tesla (TSLA) Stock Analysis — 2024

Analyzed Tesla's stock performance using Python, pandas, and matplotlib.

## What this project covers
- Price trend analysis
- 50-day & 200-day moving averages (Golden Cross identification)
- Daily returns & volatility calculation (63.6% annual volatility)
- Distribution of returns histogram

## Key Findings
- Total 2024 Return: 62.6%
- Annual Volatility: 63.6%
- Identified Golden Cross signal in Q4 2024

## Tools Used
Python | pandas | yfinance | matplotlib | numpy
